# 丁香妈妈科学养育 RAG 检索信息库

## 1. 黄疸
- **症状**: 皮肤和眼白发黄，出生后2-4天出现，高峰5-7天，约2周消退。
- **原因**: 胆红素代谢异常，生理性（正常）或病理性（感染、溶血、肝功能异常）。
- **在家护理**:
  - 增加喂奶（每2-3小时一次），促进胆红素排泄。
  - 观察黄疸程度，用黄疸仪或看皮肤颜色。
  - 适度晒太阳（隔玻璃，10-15分钟/次）。
- **就医原则**:
  - 出生24小时内出现黄疸。
  - 黄疸超2周不退。
  - 伴精神萎靡、食欲差、呕吐、腹泻。
  - 黄疸迅速加重或波及全身。
- **注意事项**:
  - 勿用中药或偏方。
  - 病理性黄疸需光疗或进一步检查。

## 2. 尿布疹
- **症状**: 臀部、腹股沟、会阴皮肤发红、丘疹，严重时糜烂、渗液、溃疡。
- **原因**: 尿液、粪便刺激，湿热环境、摩擦、过敏（尿布、湿巾）。
- **在家护理**:
  - 勤换尿布，便后用温水清洗，拍干。
  - 涂含氧化锌护臀膏。
  - 保持皮肤干爽，勿过度包裹。
- **就医原则**:
  - 皮疹超3天不退。
  - 出现糜烂、渗液、脓疱。
  - 伴发热或异常症状。
- **注意事项**:
  - 禁用含酒精湿巾。
  - 非医嘱勿用激素药膏。

## 3. 腹泻
- **症状**: 排便频繁，水样或蛋花汤样便，伴发热、呕吐。
- **原因**: 病毒（轮状病毒）、细菌、饮食不当、肠道过敏。
- **在家护理**:
  - 继续母乳/配方奶，少量多次。
  - 补充口服补液盐（ORS）防脱水。
  - 观察便性状、次数及精神状态。
- **就医原则**:
  - 腹泻超3天。
  - 伴高热、呕吐、精神萎靡、尿量减少。
  - 便带血或黏液。
- **注意事项**:
  - 勿随意用止泻药或抗生素。
  - 避免果汁或含糖饮料。

## 4. 鹅口疮
- **症状**: 口腔黏膜、舌头、上腭白色斑点，擦不掉，严重影响进食。
- **原因**: 白色念珠菌感染，奶具不洁、免疫力低、抗生素使用。
- **在家护理**:
  - 遵医嘱用抗真菌药（制霉菌素）。
  - 喂奶前后清洁宝宝口腔，奶具消毒。
  - 母乳喂养妈妈清洁乳头。
- **就医原则**:
  - 白斑持续扩散，影响进食。
  - 伴发热或异常症状。
- **注意事项**:
  - 勿强行擦拭白斑。
  - 奶瓶、奶嘴每日消毒。

## 5. 蚊虫叮咬
- **症状**: 皮肤红肿、瘙痒小包，严重时局部感染。
- **原因**: 宝宝体温高、代谢快、易出汗，吸引蚊虫。
- **在家护理**:
  - 用避蚊胺（≤30%）或埃卡瑞丁（≤20%）驱蚊液，避开眼口伤口，2个月以下禁用。
  - 保持家居卫生，用纱窗、蚊帐，穿浅色长衣裤。
  - 叮咬后涂炉甘石洗剂（无薄荷脑）。
- **就医原则**:
  - 红肿严重，面积大，持续瘙痒。
  - 出现脓疱、发热。
- **注意事项**:
  - 禁用含柠檬桉叶油、薄荷脑、樟脑产品。
  - 植物驱蚊（如橘子皮）无效。

## 6. 湿疹
- **症状**: 皮肤发红、干燥，严重渗液、结痂，多见于面部、颈部、四肢。
- **原因**: 遗传（过敏性疾病家族史）、免疫系统不成熟、皮肤屏障弱。
- **在家护理**:
  - 保持湿度50%-60%，温度22-25℃，避免出汗。
  - 穿纯棉衣物，减少摩擦。
  - 用温水（36-38℃）洗澡，2-3天一次，浴后涂无香精保湿霜（每日3-5次）。
  - 均衡饮食，勿食辛辣、海鲜。
- **就医原则**:
  - 渗液、脓疱、结痂，疑似感染。
  - 护理后无改善。
- **注意事项**:
  - 湿疹不可根治，保湿第一。
  - 中重度湿疹可用激素药膏，遵医嘱。
  - 勿盲目忌口。

## 7. 食物过敏
- **症状**: 皮肤瘙痒、口唇红肿、舌头溃烂、呕吐、打喷嚏、流涕。
- **原因**: 免疫系统不成熟，过敏原（奶、蛋、鱼、贝类、大豆、花生、坚果、小麦）。
- **在家护理**:
  - 坚持母乳喂养，4个月后适时加辅食，逐渐加量。
  - 煮熟食物，记录食物日记。
  - 停止喂食疑似过敏食物。
- **就医原则**:
  - 严重过敏症状（呼吸困难、肿胀）。
  - 症状持续或反复。
- **注意事项**:
  - 过敏原检查仅供参考，需医生评估。
  - 牛奶蛋白过敏用深度水解/氨基酸配方奶。

## 8. 胃肠胀气
- **症状**: 哭闹、蹬腿、面色胀红、屁多，严重引发肠绞痛。
- **原因**: 消化系统不成熟、吞咽空气、碳水化合物消化不良，少数由疾病/过敏。
- **在家护理**:
  - 冲奶粉避免摇晃，静置去气泡。
  - 慢速喂奶，用排气孔奶瓶，喂后拍嗝。
  - 做排气操（自行车运动）或让宝宝趴着。
- **就医原则**:
  - 哭闹不止，难以安抚。
  - 伴发热、腹泻、呕吐。
- **注意事项**:
  - 拍嗝时五指并拢、掌心内凹，轻拍背部。
  - 勿用药物治普通胀气。

## 9. 毛细支气管炎
- **症状**: 鼻塞、流涕、咳嗽、发热，2-3天后咳嗽加重、呼吸急促、喘息。
- **原因**: 病毒感染，冬春季高发，有传染性。
- **在家护理**:
  - 用生理盐水滴鼻或吸鼻器清理鼻腔。
  - 湿度40%-60%，补充水分。
  - 少食多餐，竖抱拍背。
- **就医原则**:
  - 呼吸困难、嘴唇指甲发青。
  - 脱水（尿量减少、嘴唇干）。
  - 3个月内发热或发热超3天。
- **注意事项**:
  - 坚持母乳喂养，接种流感疫苗。
  - 避免烟草，勤通风，做好隔离。

## 10. 肠绞痛
- **症状**: 持续哭闹、身体扭动、腿蜷向腹部，难以安抚。
- **原因**: 进食不足/过度、吞咽空气、牛奶蛋白过敏、妈妈吸烟/情绪焦虑。
- **在家护理**:
  - 喂奶时轻揉肚子，涂润肤霜。
  - 包裹或摇晃安抚，推婴儿车、洗热水澡、播放白噪音。
- **就医原则**:
  - 哭闹超2小时，伴呕吐、血便、体重不增。
  - 超4个月仍有肠绞痛。
- **注意事项**:
  - 妈妈避免吸烟，调整情绪。
  - 充分拍嗝，减少空气吞咽。

## 11. 幼儿急疹
- **症状**: 突发高热1-5天，退热后出皮疹，1-2天消退，少数伴腹泻、流涕。
- **原因**: 病毒感染（唾液、呼吸道分泌物传播）。
- **在家护理**:
  - 多休息，补水。
  - 遵医嘱用退热药，皮疹无需涂药。
- **就医原则**:
  - 发热超3天不退或退热后复发。
  - 3个月内发热。
  - 伴惊厥、精神萎靡、食欲差。
- **注意事项**:
  - 区分流感、风疹、麻疹、川崎病。
  - 注意热性惊厥。

## 12. 热性惊厥
- **症状**: 高烧时全身抽搐、瞪眼、身体僵直、口吐白沫，无反应。
- **原因**: 6个月-5岁儿童高烧引发，极少数导致癫痫。
- **在家护理**:
  - 放平坦柔软处，侧躺确保呼吸通畅。
  - 轻柔取出嘴内异物，记录惊厥时间。
  - 物理降温（脱衣、开空调），惊厥后就医。
- **就医原则**:
  - 惊厥超5分钟或24小时内多次发作。
  - 伴精神萎靡等症状。
- **注意事项**:
  - 勿掐人中、限制活动、塞嘴。
  - 勿用酒精擦拭降温。

## 13. 咳嗽
- **症状**: 偶尔或频繁咳嗽，伴发热、鼻塞。
- **原因**: 病毒/细菌感染、过敏、气管异物。
- **在家护理**:
  - 湿度50%-60%，用生理盐水喷鼻。
  - 6个月以上多喝温水，1岁以上适量蜂蜜（2-5ml）。
- **就医原则**:
  - 伴精神萎靡、嘴唇指甲发紫。
  - 发热超3天或咳嗽超1周无好转。
  - 呼吸急促或疑似咳嗽变异性哮喘。
- **注意事项**:
  - 2岁以下禁用非处方咳嗽药。
  - 勿滥用抗生素。

## 14. 发热
- **症状**: 体温升高（腋温≥37.2℃，肛/耳/额温≥38℃），伴精神萎靡、食欲差。
- **原因**: 多为病毒感染，少数细菌感染。
- **在家护理**:
  - 准确测体温（肛温最准）。
  - 腋温≥38.5℃或不适时用对乙酰氨基酚（3个月以上）或布洛芬（6个月以上）。
  - 保持清爽，补水，减衣。
- **就医原则**:
  - 3个月内发热。
  - 发热超3天，伴皮疹、惊厥、脱水、呼吸困难。
- **注意事项**:
  - 勿捂汗或用酒精擦拭。
  - 勿用阿司匹林、安乃近。

## 15. 鼻塞流涕
- **症状**: 流白色/黄绿色鼻涕，鼻塞，影响睡眠/进食。
- **原因**: 病毒/细菌感染、过敏性鼻炎、鼻腔异物。
- **在家护理**:
  - 用海盐水喷鼻或洗鼻器清理。
  - 保持空气湿度，多喝水。
  - 过敏性鼻炎避免过敏原，遵医嘱用鼻喷雾/抗组胺药。
- **就医原则**:
  - 症状超2周，伴发热、头痛、异味鼻涕。
  - 疑似鼻腔异物或鼻窦炎。
- **注意事项**:
  - 勿用感冒药、抠鼻子、用力擤鼻涕。

## 16. 手足口病
- **症状**: 发热，手足口肛周皮疹/水疱，口腔溃疡，伴咳嗽、流涕。
- **原因**: 病毒感染（肠道病毒），粪便、体液、呼吸道分泌物传播。
- **在家护理**:
  - 隔离2周，喂凉软食物（凉牛奶）。
  - 休息，观察症状。
- **就医原则**:
  - 高烧超48小时，伴呕吐、头痛、四肢抖动。
  - 精神萎靡、呼吸异常。
- **注意事项**:
  - 接种肠道病毒71型疫苗，勤洗手消毒。
  - 勿自行判断病情严重度。

## 17. 流感
- **症状**: 高烧（39-40℃）、咳嗽、流涕、咽痛、腹泻、全身乏力。
- **原因**: 流感病毒感染，飞沫、接触传播。
- **在家护理**:
  - 接种流感疫苗（6个月以上，8岁以下首次2剂）。
  - 勤洗手、通风、避免人多场所。
  - 确诊流感后48小时内用奥司他韦，退烧药遵医嘱。
- **就医原则**:
  - 2岁以下疑似流感。
  - 高烧伴咳嗽加重、精神萎靡。
- **注意事项**:
  - 勿信喝水10分钟、维生素C、熏醋、利巴韦林防流感。
  - 避免复方感冒药/中成药。

## 18. 痱子
- **症状**: 红痱（红色小疙瘩）、白痱（透明水疱）、脓痱（脓疱），伴瘙痒、灼痛。
- **原因**: 高温高湿、出汗不畅，胖宝宝更易发生。
- **在家护理**:
  - 保持皮肤干爽，开空调、穿棉质衣物、勤洗澡。
  - 红痱/白痱涂炉甘石洗剂，脓痱遵医嘱用抗菌药。
- **就医原则**:
  - 疑似湿疹、尿布疹等皮肤病。
  - 大片皮肤抓破或感染。
- **注意事项**:
  - 勿用十滴水、花露水（含酒精、樟脑）。
  - 脓痱禁用痱子粉。
